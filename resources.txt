Зв’язки між таблицями
------------------------------------------
Users ↔ User-role-table ↔ Role (багато-до-багатьох)
------------------------------------------
Users ↔ Class-user-table ↔ Class (багато-до-багатьох)
------------------------------------------

------------------------------------------

Ресурси та ендпоіинти

1. Users (користувачі)

GET /users/ — отримати список усіх користувачів
GET /users/{user_id} — отримати деталі конкретного користувача
POST /users/ — створити нового користувача
PUT /users/{user_id} — оновити дані користувача
DELETE /users/{user_id} — видалити користувача
------------------------------------------
2. Roles (ролі)

GET /roles/ — список ролей
GET /roles/{role_id} — деталі конкретної ролі
POST /roles/ — створити роль
PUT /roles/{role_id} — оновити роль
DELETE /roles/{role_id} — видалити роль
------------------------------------------
3. User-role (прив’язка користувача до ролі)

GET /users/{user_id}/roles/ — отримати ролі користувача
POST /users/{user_id}/roles/ — додати роль користувачу
DELETE /users/{user_id}/roles/{role_id} — видалити роль у користувача
------------------------------------------
4. Class (класи)

GET /classes/ — отримати всі класи
GET /classes/{class_id} — отримати конкретний клас
POST /classes/ — створити новий клас
PUT /classes/{class_id} — оновити дані класу
DELETE /classes/{class_id} — видалити клас
------------------------------------------
5. Class-User (зв’язок студентів із класами)

GET /classes/{class_id}/users/ — список студентів у класі
POST /classes/{class_id}/users/ — додати студента до класу
DELETE /classes/{class_id}/users/{user_id} — видалити студента з класу
------------------------------------------
6. Lesson (заняття)

GET /lessons/ — отримати всі заняття
GET /lessons/{lesson_id} — отримати конкретне заняття
POST /lessons/ — створити нове заняття
PUT /lessons/{lesson_id} — оновити дані заняття
DELETE /lessons/{lesson_id} — видалити заняття
------------------------------------------
7. Grade (оцінки)

GET /grades/ — список усіх оцінок
GET /grades/{grade_id} — отримати конкретну оцінку
POST /grades/ — поставити оцінку студенту
PUT /grades/{grade_id} — оновити оцінку
DELETE /grades/{grade_id} — видалити оцінку
------------------------------------------
8. Lesson-type (типи занять)

GET /lesson-types/ — список типів занять (предметів)
GET /lesson-types/{type_id} — отримати конкретний тип
POST /lesson-types/ — створити новий тип заняття
PUT /lesson-types/{type_id} — оновити тип заняття
DELETE /lesson-types/{type_id} — видалити тип заняття
------------------------------------------
9. Teacher-Lesson (зв’язок викладачів із предметами)

GET /teachers/{teacher_id}/lessons/ — отримати всі предмети викладача
POST /teachers/{teacher_id}/lessons/ — додати викладачу новий предмет
DELETE /teachers/{teacher_id}/lessons/{lesson_id} — видалити предмет у викладача
------------------------------------------
10. Room (аудиторії)

GET /rooms/ — список усіх аудиторій
GET /rooms/{room_id} — деталі конкретної аудиторії
POST /rooms/ — створити нову аудиторію
PUT /rooms/{room_id} — оновити дані аудиторії
DELETE /rooms/{room_id} — видалити аудиторію
